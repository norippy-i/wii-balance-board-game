バランスWiiボードを使ったゲームのコードです。

## ハードウェア
ハードウェアはVScode、Platform IOを使って書いています。
こちらご準備の上コードを動かしてもらえれば動きます

M5Stack Basic: Balance Wii board checker

Xiao: WiiBalanceboard USB HID

## ソフトウェア
ブラウザゲームですので、ローカルにダウンロードしてもらってindex.htmlを起動すれば遊べます
バランスwiiボードがなくとも、キーボードの←,→,spaceの3つの操作で動かして遊べます。
またGame over画面で"d"を押すと、スコアがリセットできます。
